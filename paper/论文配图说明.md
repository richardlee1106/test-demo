# 论文配图获取指南

本文档说明如何获取论文中标注的各个配图位置所需的图片。

## 需要准备的图片列表

### 图 1：主流标签云布局算法对比示意图

**建议内容**：展示不同布局算法（顺序布局、螺旋布局、力导向布局）生成效果的对比图
**获取方式**：

- 可从 Wordle、d3-cloud 等开源项目的演示截图获取参考
- 或使用本系统分别运行 Basic 和 Spiral 算法截图

### 图 2：动态重心引力原理示意图

**建议内容**：展示重心随标签放置动态移动的过程，包括 C₀→C₁→C₂→C₃ 的重心迁移轨迹
**获取方式**：

- 使用绘图软件（如 Visio、PPT、draw.io）手动绘制
- 包含：初始重心点、放置第一个标签、重心移动、螺旋搜索路径

### 图 3：DCGA 算法流程图

**建议内容**：算法完整流程图
**获取方式**：

- 论文中已提供 Mermaid 代码，可使用 Mermaid 在线编辑器导出
- 访问 https://mermaid.live/ 粘贴代码导出 PNG/SVG

### 图 4：径向螺旋搜索示意图

**建议内容**：以重心为原点的螺旋搜索路径示意图，标注搜索方向和碰撞检测点
**获取方式**：使用绘图软件绘制阿基米德螺旋线，标注关键参数

### 图 5：RBush 空间索引结构示意图

**建议内容**：R 树的层级结构示意图，展示最小边界矩形（MBR）的组织方式
**获取方式**：

- 参考 RBush 官方 GitHub 仓库的示意图
- 或手动绘制树形结构和空间分割图

### 图 6：两种算法布局效果对比

**建议内容**：同一组数据分别使用两种算法的布局结果对比图
**获取方式**：

1. 启动系统：`npm run dev`
2. 访问 http://localhost:5173/
3. 加载数据，分别选择 Basic 和 Spiral 算法生成标签云
4. 截取标签云区域

### 图 7：系统架构图

**建议内容**：系统四层架构（UI 层、逻辑层、计算层、数据层）
**获取方式**：

- 使用绘图软件根据论文中的 ASCII 架构图绘制正式版本
- 推荐使用 Excalidraw 或 PPT

### 图 8：地理感知标签云效果图

**建议内容**：展示标签保持地理方位关系的可视化效果
**获取方式**：

1. 启动系统
2. 选择圆形选择工具在地图上画圆
3. 选择 Geo 算法生成标签云
4. 截取标签云，展示东西南北的方位保持效果

### 图 9：布局耗时对比柱状图

**建议内容**：以标签数量为 X 轴、耗时为 Y 轴的分组柱状图
**获取方式**：

1. 启动系统
2. 访问 http://localhost:5173/benchmark/benchmark.html
3. 点击"加载数据"
4. 点击"开始测试"
5. 等待测试完成，截取"运行时间对比"图表

### 图 10：紧凑度对比折线图

**建议内容**：以标签数量为 X 轴、紧凑度为 Y 轴的折线图
**获取方式**：

- 同上，截取"紧凑度对比"图表

## 获取 Benchmark 实际数据的步骤

### 1. 启动开发服务器

```bash
cd d:\AAA_Edu\TagCloud\vite-project
npm run dev
```

### 2. 访问 Benchmark 页面

打开浏览器访问: `http://localhost:5173/benchmark/benchmark.html`

### 3. 运行测试

1. 点击 **📂 加载数据** 按钮
2. 等待所有 POI 数据加载完成（日志区域会显示加载进度）
3. 点击 **▶ 开始测试** 按钮
4. 等待测试完成（进度条会显示进度）

### 4. 获取结果

测试完成后，页面会显示：

- **统计卡片**：Basic 和 Spiral 的胜出次数、平均耗时、平均紧凑度
- **对比表格**：详细的每组配置测试结果
- **对比图表**：运行时间和紧凑度的可视化对比
- **测试结论**：自动生成的算法推荐建议

### 5. 导出报告

点击 **📄 导出报告** 按钮，可导出 Markdown 格式的完整测试报告

## 推荐绘图工具

1. **draw.io** (https://app.diagrams.net/) - 免费在线绘图，适合流程图和架构图
2. **Excalidraw** (https://excalidraw.com/) - 手绘风格图表，科研论文常用
3. **Mermaid Live Editor** (https://mermaid.live/) - 代码生成流程图
4. **ProcessOn** (https://www.processon.com/) - 国内在线绘图平台
5. **PPT/Keynote** - 简单架构图可直接使用

## 论文格式建议

### 图片格式

- 推荐使用 **PNG** 或 **PDF** 格式
- 分辨率建议 **300dpi** 以上
- 宽度建议 **800-1200px**

### 图注格式

```
图X [图片标题]
Fig. X [English Title]
```

### 表格格式

- 三线表格式
- 表头加粗
- 数据右对齐
